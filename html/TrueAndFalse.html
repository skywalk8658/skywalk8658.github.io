真與假值

Javascript裡的每一個值都可以被視為真(truchy)或假(falsy)值，物件和陣列也通常被視為真值

假值會被視為false

值/說明
【假值】
var highScore = false; 傳統布林值 false
var highScore = 0; 數字 0
var highScore = ''; NaN(不是一個數字)
var highScore = 10/'score'; 空白值
var highScore ; 一個未被值定值的變數，值為undefined
var highScore = null;
幾乎除了上述的例子外，其餘均屬於真值

真值會被視為true

【直值】
var highScore = true; 傳統布林值 true
var highScore = 1; 除了0之外的數字
var highScore = 'carrot'; 有內容的字串
var highScore = 10/5; 數字計算
var highScore = 'true'; 'true'或'false'字串
var highScore = '0'; '0'字串

檢測相等性與存在性

一個物件和陣列的存在可以被視為真值，這通常可用於檢測元件是否存在
ex:
if(document.getElementById('header')){
//元件存在: 執行一些工作任務
}else{
//元件不存在: 執行另外的工作任務
}

型別轉換

因為型別轉換，===嚴格等於和!==嚴格不等於運算子，與==等於
和!=不等於運算子相較，會發生無法預期的值機會較少

《false、0及''(空字串)使用嚴格運算子，值不相等 》
運算式\結果
(false == 0 ) => true
(false === 0 ) => false
(false == '' ) => true
(false === '' ) => false
(0 == '' ) => true
(0 === '' ) => false

《null及undefined均為假值，除了和本身比較，和其他值都是不相等的 》
運算式\結果
(undefined == null ) => true
(undefined === null ) => false
(undefined == false ) => false
(null == false ) => false
(null == 0 ) => false
(undefined == 0 ) => false

《NaN會被視為假值，與任何值都不相等，連自己都不相等(因為NaN是未定義的數值，
兩個未定義的值無法相等)》
運算式\結果
(undefined == null ) => true
(NaN == null ) => false
(NaN == NaN ) => false
